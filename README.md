# Password Strength Checker (Python CLI)

A security-focused Python command-line tool built using **Replit** to evaluate password strength and provide actionable remediation guidance.

---

## Features
- Evaluates password strength based on:
  - Length requirements
  - Uppercase and lowercase letters
  - Numbers
  - Special characters
- Uses regex-based validation
- Provides a clear strength score and rating
- Displays specific suggestions for improvement
- Designed for terminal (CLI) execution

---

## Why This Project
This project demonstrates:
- Recent hands-on experience using **Replit**
- Python CLI development and debugging
- Security hygiene and password validation concepts
- Clear, user-focused terminal output suitable for support scenarios
